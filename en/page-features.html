ESP-HACK

Firmware Features

ESP-HACK capabilities. The feature set depends on the firmware version and connected modules (CC1101 / NRF24 / IR / SD, etc.).

📶 WiFi

  • Beacon Spam - generates "junk" Wi-Fi SSIDs.
  • Evil Portal - an access point with a trap page (entered data is sent to the display).
  • Wardriving - collects network information and saves it to SD (SSID, RSSI, WPA, MAC).
  • Packets - analyzes packets of the selected network (spectrum).

🖱️ Bluetooth

  • BLE-Spam - spams BLE advertisements (iOS/Android/Windows).
  • BadBLE - emulates a keyboard over BLE (Ducky Script).
  • Mouse - when connected, emulates a mouse that moves in a circle, square, and so on.

📻 SubGHz

  • Read - receives and decodes signals (supported protocols).
  • Send - sends decoded and RAW signals. Supports `.sub` signals.
  • RAW - records RAW signals.
  • Analyzer - analyzes frequencies of remotes, keys, and so on.
  • Bruteforce - brute-forces codes for various barriers (Came/Nice/Ansonic/Holtek/Chamberlain).
  • Jammer - jamming (ILLEGAL).

📺 Infrared

  • Read - reads IR codes.
  • Send - sends IR codes. Supports `.ir` signals.
  • TV / PJ / AC - turns off TVs/projectors/air conditioners.

🧩 GPIO

    NRF24:

    • Jammer - jams 2.4-2.5 GHz (Wi-Fi, BLE, Bluetooth, USB, Video, RadioCH, ALL modes).
    • Spectrum - 2.4-2.5 GHz analyzer (signal spectrum).
    • Config - pin configuration.

    iButton:

    • Read - reads iButton keys.
    • Write - writes and rewrites keys.
