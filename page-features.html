ESP-HACK

Функции прошивки

Возможности ESP-HACK. Набор функций зависит от версии прошивки и подключённых модулей (CC1101 / NRF24 / IR / SD и т.д.).

📶 Wi‑Fi

  • Beacon Spam — генерация “мусорных” WiFi SSID.
  • Evil Portal — точка доступа со страницей-ловушкой (введённые данные отправляются на дисплей).
  • Wardriving — сбор информации о сетях и запись на SD (SSID, RSSI, WPA, MAC).
  • Packets — анализ пакетов выбранной сети (спектрум).

📱 Bluetooth

  • BLE-Spam — спам BLE рекламой (iOS/Android/Windows).
  • BadBLE — эмуляция клавиатуры по BLE (Ducky Script).
  • Mouse — при подключении эмулирует мышь, которая идет по кругу, квадрату и тд.

📻 Sub‑GHz

  • Read — приём и декодирование сигналов (поддерживаемые протоколы).
  • Send — отправка декодированных/RAW сигналов. Поддерживает .sub сигналы.
  • RAW — запись RAW сигналов.
  • Analyzer — анализ частоты пультов, ключей и тд.
  • Bruteforce — перебор кодов для разных шлагбаумов (Came/Nice/Ansonic/Holtek/Chamberlain).
  • Jammer — глушение (НЕЗАКОННО).

📺 Infrared

  • Read — чтение ИК-кодов.
  • Send — отправка ИК-кодов. Поддерживает .ir сигналы.
  • TV-OFF / PJ-OFF / AC-OFF — выключение ТВ/проектора/кондиционера.

🧩 GPIO

    NRF24:

    • Jammer — глушение 2.4-2.5 ГГц (режимы WiFi, BLE, Bluetooth, USB, Video, RadioCH, ALL).
    • Spectrum — анализатор 2.4-2.5 ГГц (спектрум сигналов).
    • Config — настройка пинов.

    iButton:

    • Read — считывание iButton ключей.
    • Write — запись/перезапись ключей.

⚙️ Настройки

  • Цвет дисплея — инвертирование цветов.
  • Standby time — режим ожидания (анимация).
  • About — информация о версии.
